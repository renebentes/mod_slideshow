mod_slideshow
=============

Módulo para exibição de artigos em Slide Show.

Requirements
------------

* [Joomla! 2.5+](http://www.joomla.org);
* [Twitter Bootstrap 2.3.2](http://www.getbootstrap.com).
* [jQuery 1.8.3+](www.jquery.com)
