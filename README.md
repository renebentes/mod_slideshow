Apresentação de Slides - mod_slideshow
=============

Módulo para exibição de artigos em uma apresentação de slides.

Requirements
------------

* [Joomla! 3.0+](http://www.joomla.org)
* [Twitter Bootstrap](http://www.getbootstrap.com).
* [jQuery](http://www.jquery.com)
